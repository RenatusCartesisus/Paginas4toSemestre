SQL, Structure Query Language(Lenguaje de Consultas Estructurado) es un lenguaje de programacion para trabajar con datos relacionales como MySQL, Oracle, etc.
MySQL es un interpretador de SQL, es un servidor de base de datos.
MySQL permite crear bases de datos y tablas, insertar datos, modificarlos, eliminarlos, ordenarlos, hacer consultas y realizar muchos operaciones. En resumen, administrar bases de datos.

Estos apuntes tienen como objetivo acercar a los conceptos iniciales para introducirse en el mundo de las bases de datos.
